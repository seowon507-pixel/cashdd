원/달러 환율봇 · 실행 안내

1. 다운로드

  1. GitHub 저장소 접속
  2. 초록색 Code 버튼 → Download ZIP
  3. 압축 해제

2. API 키 설정

.env 파일에 아래 두 값을 채워주세요.

EXIM_API_KEY=한국수출입은행 API 키 FINNHUB_API_KEY=Finnhub API 키

3. 실행 (터미널 사용)

더블클릭 실행은 macOS 보안 정책(Gatekeeper)에 막힐 수 있어, 터미널로 실행하는 것을 권장합니다.

  1. Terminal 앱 열기 (Spotlight에서 "터미널" 검색)
  2. 아래 명령어를 순서대로 입력
cd ~/Desktop/money npm install node server.js

서버가 실행되면 브라우저에서 http://localhost:3000 을 열어주세요.

종료하려면 터미널 창에서 Control + C를 누르세요.

4. 다음에 다시 실행할 때

npm install은 처음 한 번만 필요합니다. 그 다음부터는 터미널에서 아래 두 줄만 입력하면 됩니다.

cd ~/Desktop/money node server.js
